# Nota para mi cuaderno: Git y GitHub

Git es un sistema de control de versiones que registra los cambios en los archivos. GitHub es el servicio en la nube que aloja estos registros.

### Comandos Esenciales:
- `git init`: Inicializa un repositorio local.
- `git add .`: Prepara todos los archivos para el commit.
- `git commit -m "mensaje"`: Crea un punto de control con los cambios actuales.
- `git remote add origin URL`: Vincula tu carpeta local con un repositorio en GitHub.
- `git push -u origin main`: Sube tus cambios por primera vez al servidor.
- `git checkout -b nombre-rama`: Crea y entra en una nueva rama para trabajar en una función específica.
- `git merge rama`: Une los cambios de una rama a la principal.

**Flujo Típico:**
`Modificar archivo` -> `git add` -> `git commit` -> `git push`
